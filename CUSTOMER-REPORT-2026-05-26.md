# 在庫管理アプリ 改修ご報告

**対象期間**: 2026年5月24日〜26日
**最終更新**: 2026年5月26日

このたび、現場でお使いいただく上で「やりにくい」と感じられていた箇所を中心に、機能改善を行いました。主な変更点をご報告いたします。

---

## 発注のお仕事まわり

### 発注書の日付を直接書き換えられるように
- 発注日と希望納期を、画面で自由に修正していただけます

### 発注を見ながらその場で入庫処理
- 発注書を開いた画面の中で「今回○個届きました」「△個足りませんでした」を直接入力できます
- 画面を行き来する必要がなくなりました

### 不足・不良の記録が **履歴として残る**
- いつ・何個・どの理由で足りなかったかを後から確認できます
- 担当者の引き継ぎや、月次の振り返りに使えます

### 入力ミス防止
- 「発注数より入庫数の方が多い」など、ありえない数字を自動で止めます

### 【新機能】メーカー欠品時の振替発注
仕入先から「メーカー欠品で出せません」と連絡を受けた場合:
1. 該当の部品に **「別会社へ振替発注」ボタン** が表示されます
2. 別の仕入先と数量を選んで確定
3. 振替先への発注書が下書きとして自動作成されます

元の発注書 → 振替先発注書がお互いにリンクされ、後から流れを追えます。

## モノが届いた時の対応（入庫処理）

### 検査結果の選択肢を現場に合わせて整理
- 旧: 合格 / 条件付合格 / 不合格
- 新: **合格 / 不良 / 破損 / 欠品 / その他**

### 発注を探しやすく
- 工番・仕入先・納期で **検索** できるように
- 一覧から **クリック1回** で詳細を開いて操作できるように

## 工場での製造管理

### 見え方を4種類から選べます
- **一覧** : 表形式
- **カンバン** : 工程ごとに付箋を貼るような並び（直感的に動かせる）
- **ガントチャート** : 進捗が日付の横棒で一目瞭然
- **工程マスタ** : 工程そのものをカスタマイズできる

### 工程フローをお客さま自身で組み替え可能
- 工程の **追加・名前変更・並び替え・色変更・削除** がすべて画面操作で完結
- 標準: 受注 → 設計 → 板金 → 塗装 → 組み立て → 納品 / 遅延

### 製造1件ごとに「工程別」の日程を持てます
- 工程単位で開始日・納期を入力 → ガントチャートに反映
- 計画が一目で見えます

### 製造の詳細を **広い画面でじっくり** 確認
- 小さなポップアップから、フルページの専用画面に変更
- **管理者ビュー**: 工程の日程入力＋全工程チェックリスト
- **作業者ビュー**: 今やる工程を大きく表示、チェック欄も大きく

### ガントチャートの操作性向上
- 画面幅いっぱいに広がり、見やすくなりました
- 工程の棒を **ドラッグして日付を直接変更** できます
- 表示期間は 2週・1ヶ月・3ヶ月・6ヶ月・1年から選択
- 期間に応じて目盛りが日／週／月に自動最適化

### 受注の状態と製造の状態を **別々に管理**
- 営業視点（見積／受注確定／出荷済／キャンセル）
- 工場視点（受注／設計／板金／塗装／組立／納品）
- 視点ごとに混ざらず管理できるように

## 新メニュー（中身は次フェーズ）

### 製品管理表
桐の「営業 製品管理表」と同じ構成で、入力欄だけ先に並べてあります。
- 客先・工番・SNO・納期・進捗・コスト・設計・資材・鈑金塗装・組立
- 各項目はまだ「箱」のみ。保存や自動計算は次フェーズで実装予定です

このタイミングで「項目の追加／削除／並び替え」のご要望があればお伝えください。
